# Coding Reference

## if and switch Statements

### if
    if ( ( a > b ) || ( c == d ) )
    {
        // The code is here.
    }
    else if ( ( e > f ) && ( g == h ) )
    {
        // The code is here.
    }
    else
    {
        // The code is here.
    }

## Loop Statements

### for
    for ( i = 0; i < DEF_SIZE; ++i )
    {
        // The code is here.
    }

### while
    while ( loopControl == TRUE )
    {
        // The code is here.
    }

### do-while
    do
    {
        // The code is here.
    } while ( loopControl == TRUE );
